2D Graphics Editor in C — Single-File Terminal (Boilerplate Version)
A fully interactive, menu-driven 2D graphics editor that runs in a standard terminal (stdout).
It lets the user add, delete, modify, and display geometric shapes (Line, Rectangle, Circle,
Triangle)
drawn using ASCII characters ('*' for pixels, '_' for empty space) on a 80×24 character canvas.
All shapes are stored in an in-memory object array and re-rendered on every change.
"Write a 2D graphics editor in C that runs in the terminal using only standard I/O
(printf/scanf).
The program should maintain an 80×24 character canvas using a 2D array of chars.
It must support adding, deleting, and modifying 4 types of shapes:
Line (Bresenham's algorithm), Rectangle (4 line edges),
Circle (Midpoint circle algorithm), and Triangle (3 connected lines).
Each shape should be stored as an object in an array (max 100 objects), with a type
enum and up to 6 integer parameters.
Provide a text menu with options: Add Object, Delete Object, Modify Object, Display
Picture, List Objects, Exit.
After every add/delete/modify, redraw all objects from scratch onto the cleared canvas.
No external libraries — only stdio, stdlib, and math.h."

void drawLine(int x1, int y1, int x2, int y2) {
int dx = abs(x2 - x1), dy = abs(y2 - y1);
int sx = (x1 < x2) ? 1 : -1;
int sy = (y1 < y2) ? 1 : -1;
int err = dx - dy;

setPixel(x1, y1);

int e2 = 2 * err;
if (e2 > -dy) { err -= dy; x1 += sx; }
if (e2 < dx) { err += dx; y1 += sy; }

void drawRectangle(int x1, int y1, int x2, int y2) {
drawLine(x1, y1, x2, y1);
drawLine(x1, y2, x2, y2);
drawLine(x1, y1, x1, y2);
drawLine(x2, y1, x2, y2);
}
void drawCircle(int cx, int cy, int radius) {
int x = 0, y = radius;
int d = 1 - radius;
while (x <= y) {
setPixel(cx + x, cy + y); setPixel(cx - x, cy + y);
setPixel(cx + x, cy - y); setPixel(cx - x, cy - y);
setPixel(cx + y, cy + x); setPixel(cx - y, cy + x);
setPixel(cx + y, cy - x); setPixel(cx - y, cy - x);
if (d < 0) d += 2 * x + 3;
else { d += 2 * (x - y) + 5; y--; }
x++;
}
}
void drawTriangle(int x1, int y1, int x2, int y2, int x3, int y3) {
drawLine(x1, y1, x2, y2);
drawLine(x2, y2, x3, y3);
drawLine(x3, y3, x1, y1);
}
